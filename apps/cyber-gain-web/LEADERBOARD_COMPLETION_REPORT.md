## 💡 技術亮點

### 1. 動態樣式綁定
使用 Vue 的 `:style` 綁定實現複雜的條件樣式：
```vue
:style="{
  height: index < 6 ? '34px' : '40px',
  borderImage: index < 6 ? 'linear-gradient(...)' : 'none',
  background: index >= 6 ? '#202D40' : 'transparent'
}"
```

### 2. 精確的漸層邊框
使用徑向漸層實現複雜的邊框效果：
```css
border-image: radial-gradient(
  circle at 51.71% 0%,
  #FFC900 0%,
  #504311 12.64%,
  #1A2535 35.47%,
  #504311 67.20%,
  #FFC900 92.27%
) 1;
```

### 3. 內聯樣式控制
對於需要精確控制的屬性，使用內聯樣式確保優先級：
```vue
style="
  font-family: Arial, sans-serif;
  font-size: 24px;
  font-weight: 700;
"
```

### 4. 條件渲染
根據索引位置條件渲染不同的內容結構：
```vue
<div v-if="index < 6" class="flex justify-end">
  <!-- 靠右對齊 + 圖標 -->
</div>
<div v-else class="text-center">
  <!-- 居中對齊 -->
</div>
```
